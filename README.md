# vue-imitate-ele

模仿饿了么写的部分移动端页面

更大更详细 请参照 https://github.com/bailicangdu/vue2-elm

## 技术栈
vue2.x + vue-router + vuex +vue-cli 3.x + axios + es6 + mui 

## 项目运行
```
npm install

npm run serve

npm run build

npm run test

npm run lint

```
## 接口数据

感谢 bailicangdu 提供的接口数据  https://github.com/bailicangdu/node-elm/blob/master/API.md 谢谢谢谢！
